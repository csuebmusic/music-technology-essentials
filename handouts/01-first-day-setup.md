## Mac basics

Some essential shortcuts to memorize today:

| Shortcut | What it does |
|---|---|
| `Cmd + C` / `Cmd + V` | Copy / paste |
| `Cmd + Z` | Undo |
| `Cmd + S` | Save |
| `Cmd + Shift + N` | New folder (in Finder) |
| `Cmd + Space` | Spotlight (search anything) |
| `Cmd + Shift + 4` | Screenshot a region |
| `Cmd + K` (in Finder) | Connect to a server |

**Show file extensions.** Finder → Settings → Advanced → check "Show all filename extensions." This matters a lot when working with audio. You want to see `.wav`, `.aif`, `.mp3`, `.als`, etc.

## File and folder naming — the rules

This is important. Bad file names cause bad days.

- **Lowercase only.** No capital letters.
- **Hyphens, not spaces.** `my-project` not `My Project`.
- **No special characters.** No `&`, `!`, `@`, `#`, `$`, `%`, `()`, etc.
- **Be descriptive.** A name should tell you what's inside without opening it.
- **Use versions when it matters.** `lastname-soundpiece-v1.wav`, then `v2`, then `final` (and keep all versions — never overwrite).

**The format for files in this class:**

```
lastname-projectname-version.ext
```

Examples:

```
thiebaut-hello.m4a
thiebaut-soundpiece-v1.wav
thiebaut-fieldrec-traffic.wav
```

## Plugging in your gear

Your station has three things you'll connect each session:

1. **Headphones.** They plug into the **audio interface**, not the computer. (The interface is the small box on your desk with knobs and inputs.)
2. **The audio interface.** Plugged into the computer via USB. Your headphones plug into it. Microphones, when we get to recording, will plug into it too.
3. **The MIDI keyboard.** Plugged into the computer via USB. It sends note information — it doesn't make sound on its own. We'll use it later in the semester.

The lab has different models of audio interfaces and MIDI keyboards at different stations. Don't worry about model numbers — your TA will help you identify what's at your station.

### Verify in Audio MIDI Setup

After plugging things in, open **Audio MIDI Setup** (`Cmd + Space`, then type "Audio MIDI Setup"). You should see your audio interface listed. If you do, you're ready to go.

## Hello world — your first file

Here's the proof everything is working:

1. Open **QuickTime Player** (`Cmd + Space`, type "QuickTime")
2. **File → New Audio Recording**
3. Click the small dropdown arrow next to the red record button and choose your audio interface
4. Click record. Say your name and one word about why you're taking this class. Stop.
5. **File → Save**, name the file `lastname-hello.m4a`, and save it to your `lastname/week-01/` folder on the NAS

That's it. You've made your first audio file in this course.
