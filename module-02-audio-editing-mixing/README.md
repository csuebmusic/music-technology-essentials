# Module 2 — Digital Audio, Editing & Mixing in Audacity

**Weeks 2–5** (8 class meetings). Students learn how digital audio works, how to edit and shape it, and how to mix multiple tracks — culminating in **Project 1**, a short musique concrète–light piece.

## Standards

- **Audio format throughout Module 2:** 44.1 kHz / 16-bit WAV
- **DAW:** Audacity

## Key outcomes

By end of Module 2, students can:
- Explain sample rate, bit depth, aliasing, and headroom in plain language
- Record, edit, and arrange audio in Audacity
- Apply envelopes, crossfades, and basic time/pitch manipulation
- Produce a finished short piece using their own sound material
