# Module 3 — Recording, Sample Prep & Library Building

**Weeks 6–8** (6 class meetings). Students learn microphone technique, recording workflows, and how to curate a usable sample library — culminating in the **midterm**: a submitted sample library plus a terminology exam.

> **Status:** Scaffolding only — materials in development.
