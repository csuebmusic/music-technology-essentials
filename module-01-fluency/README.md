# Module 1 — Computer & Studio Fluency

**Week 1** (2 class meetings). Foundation week: get every student logged in, oriented to the lab, familiar with the file workflow, and ready for Module 2.

## Contents

- [`spec.md`](spec.md) — Module specification (goals, sessions, decisions)
- [`ta-notes.md`](ta-notes.md) — Teaching notes for the TA
- [`handouts/`](handouts/) — Student-facing reference cards
  - `01-first-day-setup.html` — Day-one login, NAS access, folder creation
  - `02-session-routines.html` — Laminated reference card at every station

## Key outcomes

By end of Week 1, every student can:
- Log into a lab Mac and locate their personal folder
- Download files from the NAS, work locally, and upload back at session end
- Identify the audio interface, headphones, and basic signal path at their station
